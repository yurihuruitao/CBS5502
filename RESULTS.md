# 模型训练与分类结果

## 1. 统一超参数

所有模型使用统一的核心超参数，确保公平对比：

| 超参 | 值 | 说明 |
|------|-----|------|
| MAX_LEN | 256 | 数据清洗阶段已过滤超过 256 token 的样本 |
| BATCH_SIZE | 32 | |
| EPOCHS | 8 | |
| PATIENCE | 3 | Early stopping 耐心值 |

各模型保留特有的学习率：BERT (3e-5)、RoBERTa (2e-5)、DeBERTa-v3 (2e-5)、BiLSTM (1e-3)、BERT Frozen MLP (1e-3)。

DeBERTa-v3-base 使用独立超参数：EPOCHS=10, PATIENCE=5, WARMUP_RATIO=0.1, WEIGHT_DECAY=0.01, BF16 混合精度（不兼容 FP16）。分类头采用 `[CLS; t1; t2; t1-t2; t1*t2]` 五路拼接 + 两层 MLP，类别权重使用 sqrt 平滑（1.31）。

## 3. 模型总体对比

| 模型 | Accuracy | Precision | Recall | Macro-F1 |
|------|----------|-----------|--------|----------|
| **Random Baseline** | 0.5000 | — | — | 0.4937 |
| **Majority Baseline** | 0.6112 | — | — | 0.3793 |
| Sentence-BERT | 0.5753 | 0.5657 | 0.5686 | 0.5646 |
| BiLSTM | 0.6276 | 0.6526 | 0.6553 | 0.6274 |
| BERT Frozen + MLP | 0.7392 | 0.7257 | 0.7262 | 0.7259 |
| RoBERTa Fine-tune | 0.7366 | 0.7294 | 0.7399 | 0.7306 |
| DeBERTa-v3 Fine-tune | 0.7552 | 0.7435 | 0.7343 | 0.7379 |
| **BERT Fine-tune** | **0.7512** | **0.7412** | **0.7500** | **0.7436** |

**关键发现：**

- **随机猜测基线（Random Baseline）** 的 Macro-F1 为 0.4937（因正负样本不平衡，略低于 0.50）；多数类基线（Majority Baseline）虽然 Accuracy 达到 0.6112，但由于完全忽略正类，Macro-F1 仅 0.3793。所有模型均显著超过这两个基线。
- **BERT Fine-tune 以 0.7436 的 Macro-F1 位列第一**，略优于 DeBERTa-v3（0.7379）和 RoBERTa（0.7306）。DeBERTa-v3 虽然在 Accuracy 上最高（0.7552），但 Macro-F1 略低于 BERT，主要因为 Recall 偏低（0.7343 vs 0.7500），说明 DeBERTa-v3 倾向于更保守的预测。三个 Transformer 微调模型差距在 1.3 个点以内，说明在相同条件下它们的实际能力接近。
- **BERT Fine-tune（0.7436）显著优于 BERT Frozen + MLP（0.7259）**，差距约 1.8 个点。微调让 BERT 学会了针对 WIC 任务优化其上下文表示，而不是仅靠通用预训练特征。
- **BiLSTM（0.6274）远高于 Sentence-BERT（0.5643）**。Sentence-BERT 使用整句级别的 mean pooling，丢失了目标词位置信息，在需要精确定位目标词语义的 WIC 任务中表现不佳；而 BiLSTM 虽然基于静态词向量，但通过 LSTM 上下文建模并精准提取目标词位置的隐状态，依然能有效区分词义。
- 所有 Transformer 微调模型的 loss 在训练过程中持续下降，但 dev F1 在 epoch 2-4 后开始停滞或下降，说明存在一定程度的过拟合，early stopping 有效防止了进一步退化。

## 4. 官方 WiC 基准测试

为验证模型的泛化能力，我们在 [官方 WiC 数据集](https://pilehvar.github.io/wic/)（Pilehvar & Camacho-Collados, 2019）上进行了评估。官方数据集包含 dev（638 样本）和 test（1400 样本）两个评测集，仅包含名词（N）和动词（V）两种词性。

| 模型 | Dev Acc | Dev F1 | Test Acc | Test F1 |
|------|---------|--------|----------|---------|
| **Random Baseline** | 0.5000 | 0.5000 | 0.5000 | 0.5000 |
| BiLSTM | 0.5361 | 0.5213 | 0.5307 | 0.5169 |
| BERT-Frozen+MLP | 0.6379 | 0.6279 | 0.6114 | 0.6025 |
| **BERT** | **0.6693** | **0.6590** | **0.6664** | **0.6584** |
| RoBERTa | 0.6661 | 0.6601 | 0.6550 | 0.6504 |
| SentenceBERT | 0.5846 | 0.5846 | 0.5229 | 0.5222 |

**分析：**

- 官方 WiC 数据集正负样本完全平衡（各 50%），因此随机猜测基线的 Accuracy 和 Macro-F1 均为 0.5000。BiLSTM 在官方数据上仅略高于随机基线（Test F1=0.5169 vs 0.5000），说明静态词向量方法在跨域场景下几乎失效。
- 所有模型在官方数据集上的表现显著低于在我们自建数据集上的表现，这符合预期——我们的模型在 SemCor 数据上训练，而官方 WiC 数据来自不同分布（WordNet 例句、VerbNet、Wiktionary 等），存在明显的 domain shift。
- **BERT 微调**在官方数据上表现最优（Test F1=0.6584），与 RoBERTa（0.6504）非常接近，两者差距仅 0.8 个点，与自建数据集上的趋势一致。
- RoBERTa 在官方数据上略低于 BERT，可能的原因是 RoBERTa 在训练数据上过拟合程度更高，泛化到不同分布的数据时表现下降。
- **BERT-Frozen+MLP（Test F1=0.6025）** 表现稳健，介于微调模型和基线之间，说明即使不微调 BERT，其预训练表示配合 MLP 也有一定的跨域泛化能力。
- BiLSTM 在官方数据上接近随机水平（~0.52），说明静态词向量方法在跨域场景下泛化能力有限。
- SentenceBERT 在 dev 上有 0.5846 但 test 上降到 0.5222，说明阈值在不同分布间迁移效果不稳定。
